# amused/contrib

Some contrib scripts.  Additions are welcome!

Scripts:

 - `amused-monitor`: a simple TUI written in perl.  It depends on:
   * p5-Curses
   * p5-Text-CharWidth
 - `amused-termux-notification`: show a persistent notification with
   buttons to control the playback under tmux (android.)
 - `amusing`: a small, experimental, GUI for amused written in TCL/TK.
